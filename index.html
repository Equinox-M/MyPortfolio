Text-to-SQL Gemini App

Transforming English Questions into SQL Queries

The "Text-to-SQL Gemini App" effortlessly transforms English queries into SQL commands using Google's Generative AI, Gemini. Built with Python and Streamlit, this intuitive web application simplifies database interactions. With SQLite managing the database, users can input natural language questions and receive accurate SQL outputs instantly. Git ensures seamless collaboration, while Markdown provides clear documentation. Streamlit enhances interactivity, creating a user-friendly experience.
